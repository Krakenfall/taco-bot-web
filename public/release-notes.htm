Taco Bot
Release Notes

Release 1.0.0

  • Split taco-bot into three different microservices, taco-bot-commands, taco-bot-web and taco-bot-news (links). This makes taco-bot easier to work on and upgrade in the future.
  • Rewrote taco-bot-commands to store its commands in MongoDB. This will make it easier to add commands in the future.
  • Updated taco-bot.com with new digs.
  • Added previews for multi-value commands on taco-bot.com, such as !jokes and !huskersermon.
  • Added new commands: roastknurdy, drama, dailyreset and carry
  • Finally added the rest of the originally submitted images for the shenanigans command.
  • Updated taco-bot-news (posts reddit links) to check for new links every few minutes instead of keeping a list of posted links. This will prevent past situations where Taco Bot vomitted 50+ links when it was out-of-date.
  • Hooked up TravisCI which will build Taco Bot deployments (all three). This is step 1 of 2 to instantly deploying Taco Bot after updates (faster updates).
  • Created support automation scripts to do build and deployment tasks that were previously de-motivating me to work on Taco Bot.
